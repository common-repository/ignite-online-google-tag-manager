Easily deploy your Google Tag Manager, Facebook Tracking Pixel, Hotjar or Hubspot tracking codes.

== Description ==

Simple Tag Manager by [Ignite Online](https://igniteonline.com.au) lets you quickly configure and deploy tracking scripts such as Google Tag Manager, Facebook Tracking Pixel, Hotjar and Hubspot.
Additional scripts can easily be added to the plugin to ensure any provider is supported.
Checkout the readme.md file included in the plugin for examples of how to extend the plugin.

== Installation ==

1. Upload the entire `ignite-tm` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

You will find 'Ignite' menu in your WordPress admin panel.
